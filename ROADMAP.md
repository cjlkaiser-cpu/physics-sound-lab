- **Atractores Acoplados** - Múltiples partículas que "dialogan"
- **Exportación MIDI** - Llevar las progresiones a un DAW

## Filosofía

Cada proyecto sigue el mismo principio:

```
Física real → Comportamiento emergente → Sonificación musical
```

No random. No algoritmos arbitrarios. La música emerge de las matemáticas.
